리다이어리는 1주일간의 독서/운동/음주량(Read, Exercise, Alcohol...-_-)을 측정해서 이번주는 제발 좀 다르게 사는데 도움을 주는 도구입니다. 물론 당신이 이 도구를 매일매일 기록할 정도로 성실하다면 당신은 이 도구가 아마 필요 없을 것입니다. 후후후..

Ruby On Rails와 Heroku를 써서 만들었습니다.

기능 : 

* 매일매일 음주/운동/독서량을 원클릭으로 편하게 기록

* 굳이 한줄일기가 쓰고싶다면 ("술 먹지 말걸" 등) 기록 가능 

* 음주/운동/독서량을 모아서 1주일 단위로 확인 (매일매일 기록만 해놨다면)

* 그래프로 음주/운동/독서량을 한 눈에
